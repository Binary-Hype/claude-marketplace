# API Documentation Generator

Invoke the api-documentation skill to create professional API documentation for Laravel applications.

## Workflow
1. Discover routes from `routes/api.php`
2. Analyze controllers, Form Requests, and API Resources
3. Build documentation with authentication, validation, and response schemas
4. Generate OpenAPI 3.0 spec, Markdown docs, and integration examples (cURL, JavaScript)

## Output
Complete endpoint documentation with request/response examples, OpenAPI/Swagger specification, error codes, and authentication guide.
